# travel

> vue2.5 仿去哪儿app项目实战

使用的技术：
 - vuex  数据状态管理
 - vue-router  前端路由
 - vue-awesome-swiper 制作轮播图
 - axios  提供数据请求
 - better-scroll  由黄轶老师编写的scroll插件
 - stylus  高效的css预编译语法
 - babel-ployfill  解决移动端部分机型白屏问题
 - fastclick 降低移动端部分机型点击延迟
 - webpack 项目打包
